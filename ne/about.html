Nevolve - Neuro-Evolution Library

Nevolve is a Neuro-Evolution Library for Reinforcement Learning, written in Python, developed aiming at enabling rapid experimentation. It implements conventional neuro-evolution algorithm, which evolves the strength of the connection weights for a fixed network topology. It initializes the Artificial Neural Network with random weights and evolves the weights using Neuro-Evolution.
